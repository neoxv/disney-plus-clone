## Installation

Install my-project with npm

```bash
  npm install 
  npm start
```
    
## Environment Variables

To run this project, you will need to add the following environment variables to your .env file. from firebase by creating a new project.

    apiKey
    authDomain
    projectId
    storageBucket
    messagingSenderId
    appId
    measurementId
